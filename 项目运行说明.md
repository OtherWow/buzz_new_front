## 如何运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
打开浏览器访问：`http://localhost:3000`

### 切换到真实后端：
当您的FastAPI后端服务准备好后，只需要：
1. 修改 `src/api/auth.ts` 中的 `useMock` 为 `false`
2. 确保后端服务运行在 `http://localhost:8000`
3. 按照 `API接口文档.md` 实现相应的接口即可
